# 🎉 v2.0.3 最终稳定版

## 版本说明

v2.0.3 是当前的**稳定版本**，所有卡片都是统一的1×1标准尺寸，确保游戏简洁、清晰、流畅。

---

## ✨ 核心特性

### 1. 金字塔式堆叠系统 🏔️NEW
- 卡片位置支持半格偏移（0.5格）
- 一张上层卡片可以同时压住多张下层卡片
- 矩形重叠检测算法
- 50%覆盖阈值判定
- 模拟真实的卡片堆叠效果

### 2. 多层堆叠系统
- 4层卡片堆叠
- 3D视觉效果
- 层级偏移展示
- 智能覆盖检测

### 3. 100%可解性保证
- 每种卡片类型数量都是3的倍数
- 自动验证并修正
- 确保每局都能完全消除

### 3. 智能卡片分组
- 相同类型自动聚集在卡槽中
- 视觉分组效果
- 更易形成三消

### 4. 7×9自适应布局
- 标准网格系统
- 自动适配容器大小
- 完美响应式设计
- 窗口大小改变自动调整

### 5. 道具系统
- 移出（3次）
- 撤销（3次）
- 洗牌（1次）
- 提示（3次）

### 6. 数据持久化
- 最高分记录
- 游戏统计
- 设置保存
