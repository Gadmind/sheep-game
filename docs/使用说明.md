## 📋 详细步骤

### 方法1：使用启动文件（推荐）

1. 找到项目文件夹：`c:\Users\29292\Desktop\sheep-game`
2. 双击 `start-python.bat` 或 `start-node.bat`
3. 等待命令窗口显示"Server started"
4. 浏览器会自动打开游戏
5. 如果没有自动打开，手动访问：`http://localhost:8000`

### 方法2：使用命令行

**打开命令行：**
- 在文件夹地址栏输入 `cmd` 然后回车

**输入命令：**
```bash
python server.py
```

**访问游戏：**
```
http://localhost:8000
```

### 方法3：直接打开（最快但功能受限）

双击 `index.html` 文件

## 🔧 如果遇到问题

### 问题1：端口8000被占用

**解决：使用其他端口**
```bash
python -m http.server 8080
```
然后访问：`http://localhost:8080`

### 问题2：Python或Node命令不存在

**解决：直接打开**
```
双击 index.html 文件
```

### 问题3：浏览器没有自动打开

**解决：手动访问**
```
在浏览器输入：http://localhost:8000
```

### 问题4：仍然有乱码

**解决：使用其他启动文件**
```
start-python.bat  （纯英文）
start-node.bat    （纯英文）
start-simple.bat  （纯英文）
```

## 📱 在手机上玩

1. **查看电脑IP**
   - 命令行输入：`ipconfig`
   - 找到IPv4地址，如：192.168.1.100

2. **启动服务器**
   - 双击 `start-python.bat`

3. **手机访问**
   - 浏览器输入：`http://192.168.1.100:8000`
   - 注意：手机和电脑要在同一WiFi
