===============================================
  HOW TO START THE GAME
===============================================

QUICKEST WAY (Recommended):
  
  Double-click one of these files:
  
  -> start-python.bat  (if you have Python)
  -> start-node.bat    (if you have Node.js)
  -> start-simple.bat  (simplest Python server)

  Then browser will open automatically!

===============================================

YOUR SYSTEM:
  Python 3.10.6 - Installed [OK]
  Node.js v18.15.0 - Installed [OK]
  
  Both are available!

===============================================

ALTERNATIVE WAYS:

1. Command Line:
   
   python server.py
   (or)
   node server.js
   (or)
   python -m http.server 8000

2. Direct Open:
   
   Double-click: index.html

===============================================

VISIT IN BROWSER:
  
  http://localhost:8000

===============================================

STOP SERVER:
  
  Press Ctrl + C in command window

===============================================

TROUBLESHOOTING:

- If port 8000 is busy, use:
  python -m http.server 8080
  
- If nothing works, just:
  Double-click index.html

===============================================

DONE! ENJOY THE GAME!
