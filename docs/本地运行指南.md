# 🎮 本地运行指南（中文）

## 🚀 最简单的启动方法

### ⭐ 推荐方式（一键启动）

**直接双击 `start.bat` 文件**

就这么简单！脚本会：
1. 自动检测您的Python或Node.js环境
2. 自动启动本地服务器
3. 自动在浏览器中打开游戏
4. 显示访问地址：`http://localhost:8000`

---

## 📋 其他启动方式

### 方式1：使用Python（已安装 ✅）

**您已安装：Python 3.10.6**

```bash
# 在项目目录打开命令行，输入：
python server.py
```

**或者使用Python内置服务器：**
```bash
python -m http.server 8000
```

**然后在浏览器访问：**
```
http://localhost:8000
```

### 方式2：使用Node.js（已安装 ✅）

**您已安装：Node.js v18.15.0**

```bash
# 在项目目录打开命令行，输入：
node server.js
```

**或者使用npm命令：**
```bash
npm start
```

**或者使用npx（推荐，支持自动刷新）：**
```bash
npx live-server --port=8000
```

**然后在浏览器访问：**
```
http://localhost:8000
```

### 方式3：直接打开（最快速）

**直接双击 `index.html` 文件**

浏览器会自动打开游戏。

⚠️ 注意：某些功能（如本地存储）在某些浏览器中可能受限。

### 完整操作流程

#### 步骤1：打开命令行

**方法A：**
1. 打开文件夹：`c:\Users\29292\Desktop\sheep-game`
2. 在地址栏输入 `cmd` 然后按回车

## 🔧 问题排查

### 问题1：端口8000已被占用

**解决方案A：使用其他端口**
```bash
python -m http.server 8080
# 然后访问 http://localhost:8080
```

**解决方案B：关闭占用端口的程序**
```bash
# 查看哪个程序占用了端口
netstat -ano | findstr :8000

# 关闭进程（替换<PID>为实际进程ID）
taskkill /PID <PID> /F
```

### 问题2：找不到python或node命令

**Python未安装？**
- 下载：https://www.python.org/downloads/
- 安装时勾选 "Add Python to PATH"

**Node.js未安装？**
- 下载：https://nodejs.org/
- 选择LTS版本

**或者直接双击 index.html 文件运行**

### 问题3：浏览器没有自动打开

手动在浏览器地址栏输入：
```
http://localhost:8000
```

### 问题4：修改代码后不生效

**强制刷新：**
```
Ctrl + F5  （Windows）
Cmd + Shift + R  （Mac）
```

## 📱 在手机/平板上访问

### 步骤1：查看电脑IP地址

```bash
# 在命令行输入
ipconfig
```

查找"IPv4 地址"，例如：`192.168.1.100`

### 步骤2：启动服务器

```bash
python server.py
# 或
node server.js
```

### 步骤3：手机访问

在手机浏览器中输入：
```
http://192.168.1.100:8000
```

**注意：**
- 手机和电脑必须在同一WiFi网络
- 防火墙可能需要允许访问
