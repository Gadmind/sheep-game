# ⭐ 立即开始游戏

---

## 🎯 三步启动

### 第一步：双击这个文件
```
start.bat
```

### 第二步：等待浏览器打开

### 第三步：开始游戏！

---

## 💡 就这么简单！

`start.bat` 会自动：
- ✅ 检测您的环境（Python/Node.js）
- ✅ 启动本地服务器
- ✅ 打开浏览器
- ✅ 显示游戏界面

**访问地址：** `http://localhost:8000`

---

## 🔧 如果start.bat不工作

### 备选方案1：使用Python
```
python server.py
```

### 备选方案2：使用Node.js
```
node server.js
```

### 备选方案3：直接打开
```
双击 index.html
```
